# 버그: 단축키 설정 시 불필요한 파일 저장 반복 발생

## 1. 발견된 이슈
- **증상**: 설정 창에서 단축키를 조작하거나 창을 이동할 때마다, 로그에 `설정 파일 저장 성공` 메시지가 과도하게 출력됨.
- **원인**: GUI 이벤트 핸들러(`on_gap_changed` 등)가 값이 변경될 때마다 매번 `config_manager.save_config()`를 호출하여 불필요한 디스크 I/O를 유발하고 있음.
- **영향**: 
  - 디스크 I/O 부하 증가.
  - 설정 파일 무결성 위험.
  - 로그 파일 용량 비정상적 증가.
