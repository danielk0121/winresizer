## 개요
- "기본값으로 초기화" 버튼 클릭 시 즉시 파일에 저장되는 현상 수정.
- "전체 단축키 삭제" 및 "기본값으로 초기화" 모두 화면 UI 상에서만 변경 사항을 보여주고 (음영 처리), 실제 `config.json` 파일 저장은 사용자가 "저장 및 적용" 버튼을 눌렀을 때만 이루어지도록 한다.

## 작업 내용
1. `web_server.py`: `/api/config/reset` API가 파일을 저장하지 않고 기본 설정값만 반환하도록 수정.
2. `app.js`: `resetConfig` 함수에서 서버에 저장 요청을 보내는 것이 아니라, 서버로부터 기본값을 받아와서 로컬 `config` 객체만 업데이트하고 UI를 렌더링하도록 수정.
3. `app.js`: "전체 삭제"와 "초기화" 시 모든 행에 `.modified` 음영이 나타나도록 보장.
