# 기능 명세: 스마트 사이클 (Smart Cycle)

## 1. 기능 개요
- 창의 크기를 단축키 반복 입력을 통해 자연스럽게 순환(Cycle)시키며 변경하는 편의 기능.
- `spec/요구사항명세서.md`에 공식적인 생산성 향상 기능으로 편입됨.

## 2. 동작 방식
- 특정 방향(예: 좌측 절반, 우측 절반) 단축키를 반복해서 입력할 경우, 창의 너비가 다음 순서로 순환하며 변경됨.
- **순환 비율**: 1/2 (절반) -> 1/3 -> 2/3 -> 1/2 (절반)

### 예시 (좌측 절반 단축키 `⌥⌘←` 입력 시)
1. 첫 번째 입력: 창이 좌측 절반(1/2) 크기로 이동 및 조절됨.
2. 두 번째 연속 입력: 창이 좌측 1/3 크기로 축소됨.
3. 세 번째 연속 입력: 창이 좌측 2/3 크기로 확장됨.
4. 네 번째 연속 입력: 다시 창이 좌측 절반(1/2) 크기로 복구됨.

## 3. 구현 상세
- `app/src/gui.py` 내의 `execute_window_command` 또는 `app/src/core/window_controller.py` 모듈에서 현재 창의 크기와 위치를 검사하여 `is_nearly_equal` 함수를 통해 다음 모드로 전환을 수행함.
- 사용자가 설정한 기본 단축키와 결합되어 추가적인 단축키 설정 없이 직관적으로 세밀한 창 크기 조절이 가능함.
