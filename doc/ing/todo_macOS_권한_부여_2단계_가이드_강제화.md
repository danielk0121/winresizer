# 작업 목록: macOS 권한 부여 2단계 가이드 강제화 정책 반영

## 개요
일부 환경에서 '손쉬운 사용' 권한만으로 작동하더라도, 모든 사용자의 환경(macOS 버전 및 보안 패치 상황)에서 100% 안정성을 보장하기 위해 **웹앱 UI에서 반드시 '손쉬운 사용'과 '입력 모니터링' 2개 권한을 모두 획득하도록 가이드**한다.

## 핵심 정책 및 구현 사항
1.  **권한 획득 필수화**: 웹앱 가이드 화면에서 '손쉬운 사용'뿐만 아니라 '입력 모니터링' 권한까지 모두 승인된 상태가 되어야만 설정 가이드를 완료한 것으로 간주한다.
2.  **웹앱 UI 가이드 개편**: 
    - [x] 가이드 단계를 1단계(손쉬운 사용) -> 2단계(입력 모니터링)로 명확히 분리하여 노출.
    - [x] 각 단계 완료 시 체크 표시(취소선 등)를 통해 사용자가 다음 행동을 인지하도록 유도.
3.  **동적 설정 버튼**: 현재 승인되지 않은 권한 중 우선순위가 높은 설정창(손쉬운 사용 우선)을 자동으로 열어주도록 버튼 로직을 개선한다.
4.  **앱 재실행 안내**: 모든 권한 승인 후 반드시 앱을 종료 후 다시 실행해야 권한이 완벽히 반영됨을 사용자에게 강력하게 안내한다.
