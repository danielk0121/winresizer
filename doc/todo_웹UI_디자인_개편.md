## 목적

현재 설정 페이지의 섹션 순서와 단축키 배치가 직관적이지 않다.
사용 빈도와 중요도 기준으로 레이아웃을 재배치한다.

## 현재 순서

1. 창 간격 (Gap)
2. 단축키 (순서 혼재)
3. 커스텀 비율 창 조절
4. 저장 및 적용 버튼 (하단)
5. 전체 단축키 초기화 버튼 (하단)

## 목표 순서

1. **저장 및 적용 버튼 + 전체 단축키 초기화 버튼** (최상단)
2. **커스텀 비율 창 조절** (좌/우/상/하 독립 행)
3. **단축키 설정** (아래 순서로 정렬)
   - 1/2 분할: 좌 → 우 → 상 → 하
   - 1/3 분할: 좌 → 중앙 → 우
   - 2/3 분할: 좌 → 우
   - 최대화
   - 리스토어
4. **창 간격 (Gap)** (최하단)

## 단축키 목록 렌더링 순서 정의

현재 `renderHotkeys()`는 `config.shortcuts` 객체 키 순서대로 렌더링한다.
명시적 순서 배열을 정의하여 고정 순서로 렌더링하도록 변경한다.

```js
const HOTKEY_ORDER = [
    'Left', 'Right', 'Top', 'Bottom',
    'Left 1/3', 'Center 1/3', 'Right 1/3',
    'Left 2/3', 'Right 2/3',
    'Top Left 1/4', 'Top Right 1/4', 'Bottom Left 1/4', 'Bottom Right 1/4',
    'Maximize', 'Restore',
];
```
