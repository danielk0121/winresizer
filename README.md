# WinResizer

윈도우 창 크기 조절기 프로젝트입니다. macOS 환경에서 창을 효율적으로 배치하고 관리할 수 있도록 도와줍니다.

## 프로젝트 구조

```
winresizer/
├── app/                             # 앱 소스 및 테스트
│   ├── src/                         # 파이썬 소스 코드
│   │   ├── core/                    # 핵심 로직 (창 관리, 설정, 단축키)
│   │   ├── ui/                      # 트레이 아이콘 및 리소스
│   │   ├── static/                  # 웹 UI 정적 파일 (JS, CSS, Favicon)
│   │   └── templates/               # 웹 UI 템플릿 (HTML)
│   └── tests/                       # TDD 기반 테스트 코드
├── doc/                             # 설계 문서 및 작업 목록 (TODO/Done)
├── spec/                            # 요구사항 명세서
├── ref/                             # 참고 이미지 및 외부 앱 분석 자료
├── build.sh                         # .app 번들 및 DMG 빌드 스크립트
└── WinResizer.spec                  # PyInstaller 빌드 설정
```

### 주요 진입점

- `app/src/main.py`: 앱 진입점 — TrayApp 실행 및 전체 컴포넌트 초기화
- `app/src/tray_app.py`: macOS 메뉴바 트레이 앱 (rumps 기반)
- `app/src/web_server.py`: 설정 및 제어를 위한 Flask 기반 웹 서버

## 실행 중 프로세스 구성

앱 실행 시 **3개의 주요 컴포넌트**가 유기적으로 동작합니다.

| # | 프로세스 / 스레드 | 역할 | 비고 |
|---|---|---|---|
| 1 | **메인 스레드** (rumps TrayApp) | macOS 메뉴바 UI 및 앱 생명주기 관리 | macOS AX API(손쉬운 사용) 호출 담당 |
| 2 | **HotkeyListenerThread** | 글로벌 키보드 단축키 감지 (pynput) | 백그라운드 상시 대기 및 창 조절 명령 트리거 |
| 3 | **Flask 웹 서버 스레드** | 브라우저 기반 설정 UI 및 API 제공 | 포트는 40000번대 랜덤 할당 (no-cache 적용) |

### 런타임 정보 및 로그 관리

- **런타임 정보 기록**: 실행 시 `PID`, `포트 번호`, `KST ISO` 형식의 시작 시간을 기록하여 프로세스 관리 및 추적성을 강화했습니다.
- **로그 롤링**: 로그 파일 용량 비대화를 방지하기 위해 날짜/시간별로 로그 파일을 생성하고 롤링하는 기능을 지원합니다.
  - 경로: `~/Library/Application Support/WinResizer/log/winresizer_YYYYMMDD_HHMMSS_KST.log`
- **설정 파일**: 사용자 맞춤 설정은 `~/Library/Application Support/WinResizer/config.json`에 저장됩니다.

## 주요 기능 및 특징

### 웹 기반 설정 UI (Modern & Intuitive)
- **실시간 피드백**: 설정을 변경하거나 삭제할 때 타임스탬프를 포함한 상세 안내 메시지를 제공합니다.
- **시각적 강조**: 설정 변경 시 해당 항목에 **노란색 음영(Highlight)** 처리를 하여 변경 사항을 즉시 확인할 수 있습니다.
- **캐시 방지 및 브랜딩**: `no-cache` 헤더를 적용하여 설정 지연 문제를 해결하였으며, `tray_icon.png`를 파비콘으로 등록하여 웹 UI의 완성도를 높였습니다.

### 정밀한 윈도우 제어
- **배치 로직 개선**: 작은 창을 우측이나 하단에 정렬할 때 화면 밖으로 나가는 버그를 수정하였으며, 사이클링 판정 로직을 개선하여 부드러운 이동을 지원합니다.
- **멀티 모니터 지원**: 활성 창이 위치한 모니터를 자동 감지하며, 모니터 간 창 이동 기능을 지원합니다.
- **커스텀 비율**: 1/2, 1/3, 2/3 등 표준 분할 외에도 사용자 정의 비율(예: 60%, 70%)에 따른 배치가 가능합니다.

## 사용 가능한 기본 단축키

설정 페이지에서 자유롭게 변경 가능합니다.

| 분류 | 기능 | 기본 단축키 |
|---|---|---|

## 설치 및 빌드 방법

### 1. 개발 환경 설정
```bash
python3 -m venv app/venv
source app/venv/bin/activate
pip install -r app/requirements.txt
```

### 2. 실행
```bash
app/venv/bin/python3 app/src/main.py
```

### 3. 패키징 (DMG 빌드)
```bash
./build.sh
```
빌드 완료 후 `dist/WinResizer.dmg` 파일을 통해 설치할 수 있습니다.

## 필수 권한 설정
macOS 시스템 설정에서 다음 권한을 허용해야 정상적으로 작동합니다.
1. **개인정보 보호 및 보안 > 손쉬운 사용**: 창 크기 및 위치 조절 권한
2. **개인정보 보호 및 보안 > 입력 모니터링**: 글로벌 단축키 감지 권한
