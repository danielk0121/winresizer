# WinResizer

WinResizer는 macOS 환경에서 윈도우 창의 크기와 위치를 효율적으로 관리할 수 있도록 설계된 강력한 창 조절 도구입니다. 복잡한 단축키 설정 없이도 직관적인 조작과 스마트한 사이클링 기능을 통해 화면을 최적으로 활용할 수 있습니다.

## 주요 특징

- **스마트 사이클링 (Smart Cycling)**: 동일한 단축키를 반복해서 누르면 창의 크기가 단계적으로 변화하여 최적의 위치를 찾습니다.
- **멀티 모니터 완벽 지원**: 활성 창이 위치한 모니터를 자동으로 감지하며, 모니터 간 창 이동 기능을 지원합니다.
- **다양한 분할 레이아웃**:
  - **기본 분할**: 1/2 (상하좌우), 1/4 (모서리)
  - **정밀 분할**: 1/3 (좌중우), 2/3 (좌우)
  - **커스텀 비율**: 사용자 정의 비율 (예: 60%, 70%) 지원
- **자동 레이아웃 (Auto Layouts)**: 특정 앱(예: Slack, iTerm2) 실행 시 미리 설정된 위치로 자동 배치합니다.
- **간격(Gap) 조정**: 창 사이의 간격을 미세하게 조정하여 시각적 편안함을 제공합니다.
- **웹 기반 설정 UI**: 브라우저를 통해 직관적으로 단축키와 앱 설정을 관리할 수 있습니다.

## 실행 아키텍처

WinResizer는 3개의 주요 컴포넌트가 유기적으로 동작하는 멀티스레드 구조입니다.

| 컴포넌트 | 역할 | 실행 환경 |
|---|---|---|
| **TrayApp (메인)** | macOS 메뉴바 UI 및 앱 생명주기 관리 | 메인 스레드 (rumps 기반) |
| **Hotkey Listener** | 글로벌 단축키 감지 및 창 제어 명령 하달 | 백그라운드 스레드 (pynput) |
| **Web Server** | 설정 UI 제공 및 REST API 처리 | 백그라운드 스레드 (Flask, 랜덤 포트) |

### 주요 API 엔드포인트
- `GET /api/config`: 현재 설정 조회
- `POST /api/config`: 설정 저장 및 즉시 반영
- `POST /api/execute`: 창 조절 명령 수동 실행

## 설치 및 실행 가이드

### 1. 개발 환경 설정 (가상 환경)
```bash
# 가상 환경 생성 및 진입
python3 -m venv app/venv
source app/venv/bin/activate

# 의존성 설치
pip install -r app/requirements.txt
```

### 2. 앱 실행
```bash
app/venv/bin/python3 app/src/main.py
```
실행 후 메뉴바의 아이콘을 클릭하여 **Preferences...**를 선택하면 설정 페이지로 이동합니다.

### 3. 패키징 (독립 실행형 앱)
```bash
./build.sh
```
`dist/WinResizer.dmg` 파일이 생성되어 일반 앱처럼 설치할 수 있습니다.

## macOS 필수 권한 설정 (2단계 가이드)

WinResizer가 정상 동작하려면 macOS의 보안 권한 승인이 필수적입니다. 앱 실행 시 웹 UI를 통해 단계별 가이드를 제공합니다.

1.  **1단계: 손쉬운 사용 (Accessibility)**
    *   창의 크기와 위치를 물리적으로 조정하기 위해 필요합니다.
    *   **주의**: 권한 허용 시 macOS 정책에 따라 앱이 자동 종료될 수 있으며, 이는 정상입니다. 재실행해 주세요.
2.  **2단계: 입력 모니터링 (Input Monitoring)**
    *   시스템 전역에서 설정한 단축키를 감지하기 위해 필요합니다.
3.  **상태 확인**: 웹 UI 상단의 배지가 **"권한 승인됨"** (초록색)으로 표시되면 모든 기능을 사용할 수 있습니다.

## 로그 및 설정 관리

- **로그 파일**: `~/Library/Application Support/WinResizer/log/` 경로에 날짜별로 저장됩니다.
- **설정 파일**: `~/Library/Application Support/WinResizer/config.json`에 사용자 설정이 보관됩니다.

## 자주 묻는 질문 (FAQ)

**Q: 특정 앱에서 창 조절이 작동하지 않습니다.**
**A:** 시스템 설정이나 일부 보안 앱은 macOS 보안 정책상 외부 제어를 차단할 수 있습니다. 그 외 일반 앱에서 안 된다면 '손쉬운 사용' 권한을 껐다 켜보세요.

**Q: 단축키를 변경했는데 바로 적용되나요?**
**A:** 네, 웹 UI에서 설정을 저장하는 즉시 백그라운드 리스너가 재시작되어 새 단축키가 반영됩니다.
