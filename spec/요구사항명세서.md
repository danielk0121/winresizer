# 요구사항 명세서

## 1. 프로젝트 개요
macOS 환경에서 전역 단축키를 이용해 활성 윈도우의 크기와 위치를 자유롭게 조절하는 생산성 유틸리티입니다. Magnet 앱과 유사한 사용자 경험을 제공하며, GUI를 통해 단축키를 사용자가 직접 설정할 수 있습니다.

## 2. 핵심 기능 요구사항
### 2.1 윈도우 제어 모드
- **1/2 분할:** 좌측, 우측, 위쪽, 아래쪽 절반 배치
- **1/3 분할:** 좌측, 중앙, 우측 1/3 배치
- **2/3 분할:** 좌측, 우측 2/3 배치
- **1/4 분할:** 좌상, 우상, 좌하, 우하 구석 배치 (Corner Snapping)
- **특수 모드:** 최대화, 이전 상태로 복구(Restore)
- **멀티 모니터 대응:** 현재 활성 창이 위치한 모니터의 해상도와 작업 영역(MenuBar 제외)을 기준으로 좌표 산출

### 2.2 사용자 인터페이스 (GUI)
- **설정 창:** macOS 다크 모드 스타일의 PyQt5 기반 GUI 제공
- **단축키 설정:** 사용자가 버튼을 클릭하고 원하는 키 조합을 입력하여 즉시 변경 가능
- **실시간 적용:** 단축키 변경 시 백그라운드 리스너를 재시작하여 앱 재실행 없이 즉시 반영
- **스크롤 뷰:** 다양한 분할 모드 설정을 쾌적하게 관리할 수 있는 스크롤 기능

### 2.3 데이터 관리 및 실행
- **설정 저장:** `config.json` 파일을 사용하여 단축키 설정을 로컬에 영구 저장 및 로드
- **접근성 권한:** 앱 실행 시 macOS '시스템 설정 > 접근성' 권한 획득 여부 자동 확인 및 안내

### 2.4 배포 및 설치
- **독립 실행형 앱 및 DMG 배포:** macOS 사용자 편의를 위한 독립 앱 번들링 및 설치용 디스크 이미지 제공 (상세: [배포 및 설치 계획서](../doc/todo_배포_및_설치.md) 참조)

## 3. 기술 스택 및 환경
- **언어:** Python 3.14+ (macOS 12.6.7 호환)
- **UI 프레임워크:** PyQt5 (macOS 버전 호환성을 위해 선정)
- **핵심 라이브러리:** `pynput` (단축키 감지), `pyobjc` (macOS Native API 제어)
- **형상 관리:** Git (Main 브랜치 중심 개발)

## 4. 제약 사항
- macOS 전용 (Apple Silicon 및 Intel Mac 지원)
- 시스템 접근성(Accessibility) 권한이 반드시 필요함
