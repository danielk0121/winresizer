# 요구사항 명세서 (요약)

## 1. 목표
macOS에서 전역 단축키를 사용해 창의 위치와 크기를 조절하는 유틸리티 개발.

## 2. 핵심 기능
- **단축키 감지:** 백그라운드에서 상시 대기.
- **창 제어:** 활성 윈도우 식별 및 위치/크기 조정.
- **멀티 모니터:** 27인치(메인), 24인치(서브) 환경 대응.

## 3. 기술 스택
- Python, `pynput`, `pyobjc` (macOS Accessibility API).

## 4. 실행 방식
- 백그라운드 상주 및 `launchd`를 통한 자동 실행.
